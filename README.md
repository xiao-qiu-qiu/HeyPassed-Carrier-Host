# HeyPassed-Carrier-Host

使用 JsMacros 编写的 南方岛 自动拉人进豁免局脚本，基于 SSNG 社区环境，依赖 SSNG 的 ircChat 进行调用。

脚本核心用途：

- 通过 IRC 私聊命令远程提交拉人请求
- 自动创建队伍、邀请玩家、进入对应玩法并执行 warp
- 从而绕过豁免机制，拉人进入豁免对局

## 运行环境

使用前请确认：

- 已安装 JsMacros 模组
- 已安装release中的配套模组包 (未知原因，不加这个模组包里的模组就会有概率不拉人，等有空再找问题)
- SSNG ircChat 功能可用 **且ircChat的prefix不是".i"**
- 主机账号为豁免

## 使用方法

1. 安装 JsMacros 模组到你的客户端。
2. 将脚本文件放入 JsMacros 的脚本目录 (见下)。
3. 进入游戏，按k键进入 JsMacros 的设置GUI。
4. 在"按键"选项卡中 可以看到第一行的最右侧有一个"+"，点击"+"。
5. 此操作新建了一个脚本槽位，点击"文件"栏目下的"./"，选中你刚刚放到脚本目录里的 "豁免运兵车ssngIRC.js"
6. 点击右下角"选择"按钮，成功将脚本加入脚本槽位
7. 将脚本状态点成"开启"，再设置一个你喜欢的键位，就可以关掉 JsMacros 的设置GUI了
8. 按你刚刚设置的按键，启动脚本
9. 脚本启动后聊天栏输出 Enabled。启动成功


脚本目录在版本文件夹下的/config/jsMacros/Macros/   如:

```text
.minecraft/versions/你的游戏版本/config/jsMacros/Macros/
```

## 使用教程

### 1. 启动脚本

进入游戏后运行该脚本。脚本启用时会在聊天栏输出：

```text
[SSNG-irc 豁免运兵车] Enabled
```

再次运行同一脚本会关闭监听：

```text
[SSNG-irc 豁免运兵车] Disabled
```

### 2. 通过 IRC 发送命令

使用 SSNG 的 ircChat 私聊运兵主机账号，发送以下格式的命令：

```text
.i 玩家ID 玩法参数
```

例如：

```text
.i Steve sw1
.i Alex bw8-1
.i 某玩家 zyzz
```

### 3. 脚本执行流程

收到命令后，脚本会自动执行以下流程：

1. 校验玩法参数是否合法。
2. 检查该玩家是否已在队列中。
3. 将请求加入队列。
4. 自动检查当前队伍状态。
5. 需要时自动创建队伍。
6. 邀请目标玩家进队。
7. 玩家入队后自动进入对应玩法。
8. 执行 warp。
9. warp 完成后将当前处理玩家移出队伍。
10. 在队列为空且冷却结束后自动解散队伍。

### 4. 返回消息说明

脚本会通过 IRC 回复请求者当前状态，例如：

- 已进入队列，第 1 位或第 N 位
- 该玩家不在线或不存在
- 该玩家已有队伍
- 组队超时
- 进入玩法失败
